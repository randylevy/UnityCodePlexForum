 With yesterday's release, the IBuilderContext has been augmented to contain the BuildKey and the OriginalBuildKey as well as some other useful information.

The "typeFrom" will be available through the OriginalBuildKey as it is the key before any mapping occured.

Therefore, you don't need to create nor modify anything in the Unity source code in order to do what you intended.
