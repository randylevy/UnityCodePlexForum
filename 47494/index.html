Where is the Source Code for the Silverlight Edition?

Wiki Link: [discussion:47494]
jagi
Feb 18, 2009 at 3:11 PM
 Hi

I am using the Composite Silverlight (CAL) library which includes the Silverlight Edition of Unity (Microsoft.Practices.Unity.dll v1.2.0.0).
The Unity source code in this section does not contains the Silverlight edition. Is the source code of the Silverlight Edition available?

Cheers,
jagi
ctavares
Feb 18, 2009 at 4:18 PM
 The source code is available either from Codeplex source control or it's also shipped as part of the Unity for Silverlight download on MSDN.

There's no separate Unity for Silverlight tree - it's all the same source, but with different project files. If you look, you'll see Unity.Silverlight.sln, which is the solution that builds the Silverlight version.
