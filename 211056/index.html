HI mtabaj,

I have ported Prism 4 over to Windows Phone 7 as well as a "Unity like" dependency injection container (Prism believes it is Unity).  I took the ContainerModel DI container from the Mobile Application Blocks (http://Mobile.CodePlex.com), implemented the IUnityContainer interface on it, and pulled over Prism 4 and Unity 2 unit test performing TDD until they all passed.   I have close to 800 passing unit test on Prism and Container.   

Available at http://PasswordMgr.CodePlex.com

It is a multi-targeting application so a single code base works on Desktop, Silverlight and Phone (the Desktop version even shares the Silverlight XAML).  In addition I updated a ported Sqlite prototype so that I have a SQL database for all three platforms (using the same code-base in IsolatedStorage).
